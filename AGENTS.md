# AGENTS.md

If you are an agent working in this repository: the actual content — every skill this
repo publishes — lives under [`skills/`](skills/), one directory per skill, each with
its own `SKILL.md` and an `evals/` directory of test cases. Start there. Read
[`CONTRIBUTING.md`](CONTRIBUTING.md) before proposing changes to an existing skill or
adding a new one; it covers the review bar (PR-only, evals required) and the security
posture (no curl-pipe-bash, no skill that fetches remote instructions at run time).
